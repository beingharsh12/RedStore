# RedStore - Shopping Website

Welcome to **RedStore**, a modern and responsive e-commerce website designed to showcase and sell cutting-edge gadgets. This project is built using HTML, CSS, and JavaScript, and includes animations and interactive features to enhance the user experience.

## Features

- **Responsive Design**: The website is fully responsive and adapts to different screen sizes.
- **Navigation Bar**: Easy-to-use navigation links for Home, About, Products, Contact, and Cart pages.
- **Hero Section**: A visually appealing hero section with a call-to-action button.
- **Product Showcase**: Highlighted products with promotional offers and discounts.
- **Animations**: Smooth animations powered by the AOS (Animate On Scroll) library.
- **Support Features**: Information about worldwide shipping, easy returns, money-back guarantees, and 24/7 support.

## Project Structure

```
redStore/
├── about.html         # About page
├── cart.css           # Styles for the cart page
├── cart.html          # Cart page
├── cart.js            # JavaScript for cart functionality
├── contact.html       # Contact page
├── index.html         # Main landing page
├── script.js          # General JavaScript for the website
├── style.css          # Main stylesheet
```

## Technologies Used

- **HTML5**: For structuring the content.
- **CSS3**: For styling and layout.
- **JavaScript**: For interactivity and dynamic features.
- **Font Awesome**: For icons.
- **AOS Library**: For scroll animations.

## How to Run the Project

1. Clone the repository or download the project files.
2. Open the `index.html` file in your browser to view the website.

## External Libraries and Resources

- [Font Awesome](https://cdnjs.cloudflare.com/ajax/libs/font-awesome/7.0.1/css/all.min.css): Used for icons.
- [AOS Library](https://unpkg.com/aos@next/dist/aos.css): Used for animations.
